This repo was made for an introductory statistics project for Northwestern's Stat 210 class.

The purpose is to analyze the leading digits of a data set to see how well (if) they follow Benford's law ([https://en.wikipedia.org/wiki/Benford%27s_law](url)).
If you can run python on your machine, simply clone the repo, and run graph_generator.py. You then input the 9 frequencies separated by spaces, and the 
title for the graph you would like. It will save a file to the same directory with the plot, titled "distribution.png".

baseball_calculation.py is the script I wrote to pull data for my project--first digit of home runs hit by qualified hitters from 2015-2025. Use it for inspiration,
or don't.

Happy graphing!
